Política de Privacidade – Petrus Rewards

Esta Política de Privacidade explica como o Petrus Rewards, o programa de recompensas do Grupo Perin, coleta, utiliza, armazena e protege as informações de seus usuários. Ao utilizar o aplicativo, você concorda com os termos descritos abaixo.

1. Dados Coletados

O aplicativo Petrus Rewards coleta os seguintes dados pessoais e informações relacionados às notas fiscais:

  • Dados Pessoais:
    • Nome
    • E-mail
    • CPF
  • Notas Fiscais:
    • Foto da nota fiscal
    • Número da nota fiscal
    • Código da nota fiscal
    • Data e hora da emissão
    • Valor da compra

2. Finalidade da Coleta

Os dados coletados são utilizados exclusivamente para:

  • Identificar e autenticar o usuário.
  • Verificar a autenticidade das notas fiscais emitidas em lojas do Grupo Perin.
  • Atribuir pontos ao usuário, que podem ser utilizados para concorrer a prêmios ao final do ano.

Importante:

  • Não utilizamos dados para envio de notificações push.
  • Não utilizamos dados para personalização de marketing.

3. Armazenamento e Segurança

Os dados são armazenados de forma segura utilizando os serviços do Firebase Auth e Firebase Realtime Database. As medidas de segurança implementadas seguem as melhores práticas recomendadas pela documentação oficial do Firebase, garantindo proteção contra acessos não autorizados e vazamentos de informações.

4. Compartilhamento de Dados

Os dados coletados são acessados pelo Grupo Perin exclusivamente para:

  • Confirmar a autenticidade das notas fiscais.
  • Atribuir prêmios aos usuários ao final do ano, de acordo com os pontos acumulados.

Importante: Os dados não são compartilhados com terceiros para finalidades de marketing ou qualquer outro propósito fora do contexto do programa de recompensas.

5. Direitos dos Usuários

Os usuários têm os seguintes direitos:

  • Exclusão de Dados: A exclusão dos dados pessoais pode ser solicitada entrando em contato diretamente com o Grupo Perin, conforme especificado nos Termos de Uso do aplicativo.
  • Correção de Dados: Alterações ou atualizações podem ser solicitadas pelo mesmo canal.

6. Tecnologias de Rastreamento

O aplicativo utiliza as seguintes tecnologias de rastreamento:

  • Firebase Analytics: Para monitorar o desempenho do aplicativo.
  • Crashlytics: Para identificar e corrigir falhas técnicas.

Esses serviços são utilizados apenas para melhoria contínua do aplicativo e não capturam dados para fins de marketing.

7. Alterações na Política de Privacidade

Podemos atualizar esta Política de Privacidade periodicamente. Em caso de alterações, os usuários serão notificados por meio do aplicativo. Recomendamos revisar esta política regularmente para estar ciente de possíveis mudanças.
